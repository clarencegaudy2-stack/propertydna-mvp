RECOMMENDATION SYSTEM STANDARDIZATION
==================================================

Current issue:

Deal Rating and Recommendation can display conflicting language.

Example:

Rating:
REJECT

Recommendation:
PASS

This creates confusion.

Replace current recommendation logic with:

IF Deal Score >= 75

Deal Rating:
STRONG DEAL

Recommendation:
PROCEED

Color:
Green

--------------------------------------------------

IF Deal Score >= 50 AND Deal Score < 75

Deal Rating:
REVIEW

Recommendation:
ANALYZE FURTHER

Color:
Yellow

--------------------------------------------------

IF Deal Score < 50

Deal Rating:
REJECT

Recommendation:
DO NOT PROCEED

Color:
Red

--------------------------------------------------

All report screens, dashboard cards, and deal detail pages must use the same terminology.
