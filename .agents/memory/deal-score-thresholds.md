## Thresholds
- **≥ 75**: Strong Deal / Proceed / Green
- **50 – 74**: Review / Analyze Further / Yellow  
- **< 50**: Reject / Do Not Proceed / Red

**Why:** Established in the original calculator design. All results page UI (score badge color, recommendation text, action button label) must be consistent with these bands.

**How to apply:** Any new scoring display, filter, or export must use these exact cutoffs.
